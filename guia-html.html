Etiquetas HTML usadas en el Portafolio

Cada etiqueta que aparece aquí fue usada en tu código real.

<!DOCTYPE html>

No es una etiqueta, es una declaración. Le dice al navegador que el documento usa HTML5. Siempre va en la primera línea.

<!DOCTYPE html>

<html>

Es la raíz del documento. Todo el contenido de la página va dentro de ella. El atributo lang="es" indica el idioma de la página.

<html lang="es"> ... </html>

<head>

Contiene información del documento que el usuario no ve directamente: el título de la pestaña, los estilos, las fuentes y la configuración de pantalla.

<head> ... </head>

<meta charset="UTF-8">

Define la codificación de caracteres. Con UTF-8 la página puede mostrar tildes, ñ y símbolos especiales correctamente.

<meta charset="UTF-8">

<meta name="viewport">

Hace que la página se vea bien en celulares y tablets. Sin esta línea el diseño se vería muy pequeño en móviles.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

<title>

Define el texto que aparece en la pestaña del navegador.

<title>Portafolio</title>

<link>

Conecta recursos externos al documento. Se usa para cargar el archivo CSS y las fuentes de Google Fonts. El atributo rel indica el tipo de relación y href la dirección.

<link rel="stylesheet" href="style.css">

<body>

Todo lo que el usuario ve en pantalla va dentro del body: textos, imágenes, botones, etc.

<body> ... </body>

<header>

Representa la cabecera de la página. Normalmente contiene el logo y la navegación. Es una etiqueta semántica, lo que significa que le dice al navegador qué tipo de contenido hay ahí.

<header> ... </header>

<div>

Es un contenedor genérico. No tiene significado propio pero sirve para agrupar elementos y aplicarles estilos. Es uno de los más usados en HTML.

<div class="logo">JEIMAR RUBIANO</div>

<nav>

Etiqueta semántica para la barra de navegación. Indica que dentro hay enlaces para moverse entre páginas.

<nav> ... </nav>

<a>

Crea un enlace. El atributo href indica a dónde lleva. Con target="_blank" el enlace abre en una pestaña nueva.

<a href="flexbox.html">FLEXBOX</a>

<main>

Contiene el contenido principal de la página. Solo debe haber uno por página. Es semántico y ayuda a la accesibilidad.

<main> ... </main>

<section>

Agrupa contenido relacionado dentro del main. Se usa para dividir la página en bloques con sentido propio.

<section> ... </section>

<article>

Representa un bloque de contenido independiente, como una tarjeta o una publicación. Podría existir por sí solo fuera de la página.

<article class="tarjeta"> ... </article>

<img>

Muestra una imagen. src es la dirección de la imagen y alt es el texto alternativo que se muestra si la imagen no carga, también usado por lectores de pantalla.

<img src="foto.jpg" alt="Jeimar">

<p>

Define un párrafo de texto. El navegador añade un salto de línea automático antes y después.

<p class="descripcion">Soy programador en formación.</p>

<footer>

Representa el pie de página. Normalmente contiene derechos de autor, contacto o información secundaria.

<footer> ... </footer>
