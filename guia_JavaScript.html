⚡ Guía Completa de JavaScript

JavaScript hace que las páginas web sean interactivas. Con HTML pones el contenido, con CSS lo diseñas, y con JS lo haces funcionar.

1. Cómo conectar JavaScript al HTML

Igual que el CSS, hay dos formas. La mejor es un archivo separado.

<!-- FORMA 1: Archivo separado (recomendada) -->
<!-- Va al final del body, antes de </body> -->
<script src="script.js"></script>

<!-- FORMA 2: Directo en el HTML -->
<script>
    console.log("Hola mundo");
</script>

<!-- Por qué al final del body?
     Para que el HTML cargue primero y JS pueda encontrar los elementos -->
console.log() muestra mensajes en la consola del navegador. Para abrirla: F12 → pestaña "Console"

2. Variables

Guardan información. En JS moderno se usan let y const.

let nombre = "Naruto";       // let: puede cambiar su valor
const PI = 3.1416;           // const: no puede cambiar (constante)

// Tipos de datos
let texto = "Hola";          // string (texto, va entre comillas)
let numero = 25;             // number (número)
let decimal = 3.14;          // number decimal
let activo = true;           // boolean (true o false)
let vacio = null;            // null (vacío intencionalmente)
let sinDefinir = undefined;  // undefined (no tiene valor aún)

// Ver el valor en consola
console.log(nombre);         // → Naruto
console.log(typeof nombre);  // → string (dice el tipo)
Diferencia clave: usa const por defecto. Solo usa let si el valor va a cambiar. Evita var (es viejo y tiene problemas).

3. Operadores

// MATEMÁTICOS
5 + 3   // → 8   suma
5 - 3   // → 2   resta
5 * 3   // → 15  multiplicación
5 / 2   // → 2.5 división
5 % 2   // → 1   módulo (resto de la división)
2 ** 3  // → 8   potencia (2³)

// COMPARACIÓN (devuelven true o false)
5 == "5"   // → true  (compara solo el valor, no el tipo)
5 === "5"  // → false (compara valor Y tipo) ← usa siempre este
5 !== 3    // → true  (diferente)
5 > 3      // → true  (mayor que)
5 < 3      // → false (menor que)
5 >= 5     // → true  (mayor o igual)

// LÓGICOS
true && false  // → false  (AND: los dos deben ser true)
true || false  // → true   (OR: al menos uno debe ser true)
!true          // → false  (NOT: invierte el valor)

// ASIGNACIÓN
let x = 5;
x += 3;   // x = x + 3 → 8
x -= 2;   // x = x - 2 → 6
x *= 2;   // x = x * 2 → 12
x++;      // x = x + 1 → 13
x--;      // x = x - 1 → 12

4. Strings - trabajar con texto

const nombre = "Naruto";
const apellido = "Uzumaki";

// Unir strings
const completo = nombre + " " + apellido;  // "Naruto Uzumaki"

// Template literals (la forma moderna, usa backticks `)
const saludo = `Hola, soy ${nombre} ${apellido}`;
// → "Hola, soy Naruto Uzumaki"

// Métodos útiles
nombre.length           // → 6       (cantidad de caracteres)
nombre.toUpperCase()    // → "NARUTO" (todo mayúsculas)
nombre.toLowerCase()    // → "naruto" (todo minúsculas)
nombre.includes("aru")  // → true     (contiene ese texto?)
nombre.startsWith("Na") // → true     (empieza con?)
nombre.endsWith("to")   // → true     (termina con?)
nombre.replace("Naruto", "Sasuke") // → "Sasuke"
nombre.trim()           // quita espacios al inicio y al final
nombre.split("")        // divide en array de caracteres
"  hola  ".trim()       // → "hola"

// Acceder a un carácter
nombre[0]               // → "N" (primer carácter)
nombre[nombre.length-1] // → "o" (último carácter)

5. Condicionales - tomar decisiones

// IF / ELSE IF / ELSE
const edad = 17;

if (edad >= 18) {
    console.log("Eres mayor de edad");
} else if (edad >= 13) {
    console.log("Eres adolescente");
} else {
    console.log("Eres niño");
}

// OPERADOR TERNARIO (if/else en una línea)
const mensaje = edad >= 18 ? "Mayor" : "Menor";
// si la condición es true → "Mayor", si no → "Menor"

// SWITCH (cuando hay muchas opciones)
const dia = "lunes";
switch (dia) {
    case "lunes":
        console.log("Inicio de semana");
        break;
    case "viernes":
        console.log("Fin de semana!");
        break;
    default:
        console.log("Día normal");
}

6. Bucles - repetir código

// FOR: cuando sabes cuántas veces repetir
for (let i = 0; i < 5; i++) {
    console.log(i);  // → 0, 1, 2, 3, 4
}

// WHILE: mientras una condición sea true
let contador = 0;
while (contador < 3) {
    console.log(contador);
    contador++;
}

// FOR...OF: recorrer un array
const personajes = ["Naruto", "Sasuke", "Sakura"];
for (const personaje of personajes) {
    console.log(personaje);
}

// FOR...IN: recorrer las claves de un objeto
const ninja = { nombre: "Naruto", aldea: "Hoja", rango: "Hokage" };
for (const clave in ninja) {
    console.log(clave, ":", ninja[clave]);
}

// FOREACH: método de array (más moderno)
personajes.forEach(p => console.log(p));

7. Funciones

// FUNCIÓN NORMAL
function saludar(nombre) {
    return `Hola, ${nombre}!`;
}
console.log(saludar("Naruto")); // → "Hola, Naruto!"

// FUNCIÓN FLECHA (arrow function, forma moderna)
const saludar = (nombre) => `Hola, ${nombre}!`;

// Si tiene una sola línea, el return es automático
const doble = n => n * 2;
console.log(doble(5)); // → 10

// Si tiene varias líneas, necesita llaves y return
const sumar = (a, b) => {
    const resultado = a + b;
    return resultado;
};

// PARÁMETROS POR DEFECTO
function saludar(nombre = "visitante") {
    return `Hola, ${nombre}!`;
}
saludar();          // → "Hola, visitante!"
saludar("Naruto");  // → "Hola, Naruto!"

8. Arrays - listas de datos

const frutas = ["manzana", "pera", "uva"];

// Acceder
frutas[0]           // → "manzana"
frutas.length       // → 3

frutas.push("kiwi")     // agrega al final

frutas.unshift("fresa") // agrega al inicio
frutas.shift()          // quita el primero

// Buscar
frutas.includes("pera")     // → true
frutas.indexOf("uva")       // → 2 (posición)

// Recorrer y transformar
frutas.forEach(f => console.log(f))  // recorre sin devolver nada

const mayusculas = frutas.map(f => f.toUpperCase())
// → ["MANZANA", "PERA", "UVA"]  (crea un array nuevo)

const largas = frutas.filter(f => f.length > 4)
// → ["manzana"] (filtra los que cumplen la condición)

const total = [1,2,3,4].reduce((acc, n) => acc + n, 0)
// → 10 (acumula un valor)

// Ordenar
frutas.sort()           // ordena alfabéticamente
[3,1,2].sort((a,b) => a - b)  // ordena números de menor a mayor

// Unir y dividir
frutas.join(", ")       // → "manzana, pera, uva"
"a,b,c".split(",")      // → ["a", "b", "c"]

9. Objetos

// Crear un objeto
const ninja = {
    nombre: "Naruto",
    edad: 17,

ninja["aldea"]        // → "Hoja"

ninja.rango = "Hokage";

// Desestructuración (sacar propiedades en variables)
const { nombre, edad } = ninja;

// Spread (copiar o combinar objetos)
const copia = { ...ninja };
const combinado = { ...ninja, rango: "Hokage" };

10. DOM - Manipular el HTML con JS

El DOM es la representación del HTML en JS. Con él puedes cambiar texto, estilos, agregar elementos y más.

// SELECCIONAR ELEMENTOS
document.getElementById("miId")          // por id
document.querySelector(".miClase")       // por clase (el primero)
document.querySelectorAll(".miClase")    // todos los que coincidan
document.querySelector("h1")            // por etiqueta

// CAMBIAR CONTENIDO
const titulo = document.querySelector("h1");
titulo.textContent = "Nuevo título";     // cambia el texto
titulo.innerHTML = "<em>Cursiva</em>";  // cambia el HTML interno

// CAMBIAR ESTILOS
titulo.style.color = "red";
titulo.style.fontSize = "32px";
titulo.style.display = "none";          // oculta el elemento

// CAMBIAR CLASES
titulo.classList.add("activo");         // agrega clase
titulo.classList.remove("activo");      // quita clase
titulo.classList.toggle("activo");      // agrega si no está, quita si está

// CAMBIAR ATRIBUTOS
const img = document.querySelector("img");
img.src = "nueva-foto.jpg";
img.setAttribute("alt", "nueva descripción");

// CREAR Y AGREGAR ELEMENTOS
const parrafo = document.createElement("p");  // crea un <p>
parrafo.textContent = "Nuevo párrafo";
document.body.appendChild(parrafo);           // lo agrega al body

// ELIMINAR ELEMENTOS
parrafo.remove();

13. Fetch - obtener datos de una API

Sirve para pedir datos a un servidor sin recargar la página.

// Forma básica
fetch("https://api.ejemplo.com/datos")
    .then(respuesta => respuesta.json())  // convierte a JSON
    .then(datos => {
        console.log(datos);               // usa los datos
    })
    .catch(error => {
        console.log("Error:", error);     // si algo falla
    });

// Forma moderna con async/await (más fácil de leer)
async function obtenerDatos() {
    try {
        const respuesta = await fetch("https://api.ejemplo.com/datos");
        const datos = await respuesta.json();
        console.log(datos);
    } catch (error) {
        console.log("Error:", error);
    }
}

obtenerDatos();
async/await es la forma moderna de manejar operaciones que tardan tiempo (como pedir datos a internet). await "espera" a que termine antes de continuar.

14. LocalStorage - guardar datos en el navegador

Guarda datos que persisten aunque cierres el navegador. Es como el JSON pero en el navegador.

// GUARDAR
localStorage.setItem("nombre", "Naruto");
localStorage.setItem("edad", JSON.stringify(17)); // números y objetos con JSON.stringify

// LEER
const nombre = localStorage.getItem("nombre");    // → "Naruto"
const edad = JSON.parse(localStorage.getItem("edad")); // → 17

// ELIMINAR
localStorage.removeItem("nombre");
localStorage.clear(); // borra todo

// EJEMPLO PRÁCTICO: guardar preferencia de tema
document.getElementById("btn-tema").addEventListener("click", () => {
    document.body.classList.toggle("oscuro");
    const tema = document.body.classList.contains("oscuro") ? "oscuro" : "claro";
    localStorage.setItem("tema", tema);
});

// Al cargar la página, aplicar el tema guardado
const temaGuardado = localStorage.getItem("tema");
if (temaGuardado === "oscuro") {
    document.body.classList.add("oscuro");
}
