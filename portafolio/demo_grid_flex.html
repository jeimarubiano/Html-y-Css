¿Cuándo usar cada uno?

CSS Grid Flexbox
Dimensiones 2D (filas + columnas) 1D (fila O columna)
Ideal para Layouts de página completa Componentes y barras de nav
Control Desde el contenedor Contenedor + hijos
Responsivo auto-fit / minmax flex-wrap / flex-grow
Ejemplo típico Dashboard, galería, página Navbar, botones, cards
