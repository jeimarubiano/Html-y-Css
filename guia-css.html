Estilos CSS usados en el Portafolio

Cada propiedad que aparece aquí fue usada en tu archivo style.css real.

display

Define cómo se comporta un elemento en el flujo de la página.

display: flex;

Con flex el elemento se convierte en contenedor flexible y sus hijos se organizan en fila o columna. Con none el elemento desaparece completamente.

flex-direction

Define la dirección en que se acomodan los hijos dentro de un contenedor flex.

flex-direction: column;

column los pone uno debajo del otro. row (el valor por defecto) los pone uno al lado del otro.

justify-content

Alinea los hijos a lo largo del eje principal (horizontal en row, vertical en column).

justify-content: space-between;

space-between pone el primer elemento al inicio y el último al final. center los centra. flex-start los pega al inicio.

align-items

Alinea los hijos en el eje secundario (vertical en row).

align-items: center;

center centra todos los elementos verticalmente sin importar su altura. flex-start los pega arriba.

gap

Define el espacio entre los hijos de un contenedor flex o grid. Es más limpio que usar margin en cada elemento.

gap: 10px;

flex

Le dice a un hijo cuánto espacio puede ocupar dentro del contenedor flex.

flex: 1;

Con flex: 1 el elemento crece para ocupar todo el espacio disponible. Se usó en el menú para que ocupara el centro entre el logo y el logo2.

background-image

Pone una imagen de fondo en un elemento.

background-image: url(fondo.jpg);

La dirección puede ser una ruta local como fondo.jpg o una URL de internet.

background-size

Controla el tamaño de la imagen de fondo.

background-size: cover;

cover hace que la imagen cubra todo el elemento sin dejar espacios en blanco, recortando si es necesario.

background-position

Indica en qué parte del elemento se ubica la imagen de fondo.

background-position: center;

center centra la imagen tanto horizontal como verticalmente.

background-attachment

Define si el fondo se mueve o se queda fijo al hacer scroll.

background-attachment: fixed;

fixed hace que el fondo quede estático mientras el contenido se desplaza encima, creando un efecto parallax.

background-repeat

Controla si la imagen de fondo se repite cuando es más pequeña que el elemento.

background-repeat: no-repeat;

no-repeat evita que la imagen se repita.

min-height

Define la altura mínima de un elemento. El elemento puede crecer más si el contenido lo necesita.

min-height: 100vh;

100vh equivale al 100% de la altura visible de la pantalla. Se usa en el body para que la página siempre ocupe la pantalla completa.

margin

Define el espacio exterior de un elemento, es decir el espacio entre ese elemento y los de su alrededor.

margin: 0;

margin: 0 elimina el margen por defecto que tienen el body y otros elementos. margin: auto centra horizontalmente un elemento con ancho definido.

padding

Define el espacio interior de un elemento, entre su borde y su contenido.

padding: 20px 50px;

Con dos valores: el primero es arriba y abajo, el segundo es izquierda y derecha. Con padding-top se aplica solo arriba.

box-sizing

Define cómo se calcula el tamaño total de un elemento.

box-sizing: border-box;

Con border-box el padding y el borde se incluyen dentro del ancho y alto del elemento, así no se desborda. Es la opción más predecible.

width

Define el ancho de un elemento.

width: 400px;

Puede ser en píxeles px, porcentaje % o relativo al viewport vw. Con 100% ocupa todo el ancho de su contenedor.

height

Define la altura de un elemento.

height: 400px;

border-radius

Redondea las esquinas de un elemento.

border-radius: 30px;

Con 50% convierte el elemento en un círculo perfecto si tiene el mismo ancho y alto.

border

Define el borde de un elemento. Se indica el grosor, el estilo y el color.

border: 3px solid rgba(43, 22, 161, 0.281);

solid es una línea continua. none elimina el borde.

border-bottom / border-top

Aplica borde solo en un lado del elemento.

border-bottom: 2px solid purple;

box-shadow

Añade una sombra alrededor del elemento. Los valores son: desplazamiento horizontal, vertical, difuminado y color.

box-shadow: 0 8px 50px purple;

color

Define el color del texto de un elemento.

color: white;

Se puede escribir como nombre (white), hexadecimal (#f0a500) o con transparencia (rgba(255,255,255,0.7)).

background / background-color

Define el color de fondo de un elemento.

background: rgba(0, 0, 0, 0.322);

rgba permite añadir transparencia con el cuarto valor (0 = transparente, 1 = sólido).

font-size

Define el tamaño del texto.

font-size: 14px;

font-weight

Define el grosor del texto.

font-weight: bold;

bold lo pone en negrita. También se pueden usar números como 400 (normal) o 700 (negrita).

font-family

Define la fuente del texto.

font-family: "Cinzel", serif;

Se pueden poner varias fuentes separadas por coma. Si la primera no carga, el navegador usa la siguiente. serif es una fuente genérica de respaldo.

font-style

Define si el texto es normal o cursiva.

font-style: normal;

text-align

Alinea el texto dentro de su elemento.

text-align: center;

También puede ser left, right o justify.

text-decoration

Controla decoraciones del texto como subrayado.

text-decoration: none;

Se usa en los enlaces <a> para quitarles el subrayado que traen por defecto.

line-height

Define el espacio entre líneas de texto.

line-height: 1.8;

Un valor de 1.8 significa que el espacio entre líneas es 1.8 veces el tamaño del texto. Mejora la legibilidad.

white-space

Controla cómo se maneja el espacio en blanco dentro de un elemento.

white-space: nowrap;

nowrap evita que el texto haga salto de línea. Se usó en el logo para que no se parta en dos líneas.

object-fit

Define cómo se ajusta una imagen dentro de su contenedor.

object-fit: cover;

cover hace que la imagen cubra todo el espacio sin deformarse, recortando los bordes si es necesario.

position

Controla cómo se posiciona un elemento en la página.

position: relative;

relative posiciona el elemento respecto a su lugar normal. absolute lo posiciona respecto al padre con position: relative.

inset

Atajo para definir top, right, bottom y left al mismo tiempo. Solo funciona en elementos con position.

inset: 0;

inset: 0 hace que el elemento ocupe exactamente el mismo espacio que su padre posicionado.

z-index

Controla qué elemento queda encima cuando dos se superponen. El número mayor queda encima.

z-index: 1;

transition

Anima el cambio de una propiedad CSS de forma suave cuando esta cambia, por ejemplo al hacer hover.

transition: transform 0.3s ease, box-shadow 0.3s ease;

Los valores son: la propiedad que anima, la duración y el tipo de velocidad (ease empieza lento, acelera y termina lento).

transform

Aplica transformaciones visuales a un elemento sin afectar el layout.

transform: translateY(-8px);

translateY(-8px) mueve el elemento 8px hacia arriba. Se usó en el hover de las tarjetas para el efecto de elevación.

animation

Aplica una animación definida con @keyframes a un elemento.

animation: luzSuave 3s linear infinite;

Los valores son: nombre de la animación, duración, velocidad (linear = constante) y repetición (infinite = sin fin).

@keyframes

Define los estados de una animación CSS. from es el estado inicial y to el final. También se pueden usar porcentajes como 0%, 50%, 100%.

@keyframes luzSuave { from { filter: hue-rotate(0deg); } to { filter: hue-rotate(360deg); } }

filter

Aplica efectos visuales a un elemento como desenfoque, brillo o rotación de color.

filter: hue-rotate(360deg);

hue-rotate rota los colores del espectro. Se usó para animar el borde de la foto de perfil.

content

Propiedad obligatoria en los pseudo-elementos ::before y ::after. Define qué contenido insertar.

content: '';

Con '' se inserta un elemento vacío que sirve como decoración o fondo.

margin-top

Aplica margen solo en la parte superior del elemento.

margin-top: auto;

auto en el footer dentro de un flex column empuja el footer hasta el fondo de la página.
